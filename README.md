# 🖥️ Pemrograman Visual Berbasis Python

Halo! Selamat datang di repositori pembelajaran dan pengumpulan tugas untuk mata kuliah **Pemrograman Visual**. 

Repositori ini berisi kumpulan proyek antarmuka pengguna grafis (GUI) yang dibangun menggunakan bahasa pemrograman Python dan *framework* **PySide6**. Fokus utama dari proyek-proyek di sini adalah memahami konsep tata letak (*layouting*), manipulasi *widget*, penataan gaya (menggunakan QSS), dan logika *event-driven programming* (Signals & Slots).

---

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Python 3.x
* **GUI Framework:** PySide6 (Qt for Python)
* **Editor/IDE:** Visual Studio Code
